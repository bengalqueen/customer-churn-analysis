# Customer Churn Analysis
Analyzed behavior of 7,032 telecom customers to identify churn patterns 
based on tenure, contract type, and monthly charges.
## Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
## Key Analysis
- Churn Distribution
- Tenure vs Churn
- Monthly Charges vs Churn
- Contract Type vs Churn
- Internet Service vs Churn
- Payment Method vs Churn
- Senior Citizen vs Churn
- Tenure Bands vs Churn
- Correlation Heatmap
- Monthly Charges Distribution by Churn
## Key Findings
- Overall churn rate: 26.58%
- 31.84% churned users were on monthly contract with tenure <= 3 months
- Churned customers pay ₹13 more on average than retained customers
- 25.47% of churned customers are Senior Citizens
